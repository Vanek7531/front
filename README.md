## Запуск

Нужен локальный сервер — без него `fetch` не загрузит `mok.json`.

```bash
npx serve .
```

Открыть `http://localhost:3000`.

## Структура

```
index.html          — разметка
css/style.css       — стили (подключаются в HTML)
scss/               — исходники SCSS
js/                 — логика: фильтр, поиск, load more
mok.json            — данные курсов
```
## Функционал

- фильтрация по категориям
- живой поиск по названию
- подгрузка карточек (Load more)
- fluid-адаптив без жёстких брейкпоинтов (`clamp` + `auto-fit` grid)
